# RFC — forward-looking proposals

Open questions and proposals **not yet decided**. An RFC graduates to an ADR (in
`docs/decisions/`) when accepted, or is closed when rejected/obsolete.

## Naming

`YYYY-MM-DD-topic.md`.

## Status values

`Open` · `Accepted → ADR YYYY-MM-DD-…` · `Rejected` · `Deferred`
